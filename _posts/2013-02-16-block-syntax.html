Remember this ya'll!

Method parameters. namedParameter: (t(^) (t1,...,tn)) variableName

Type definitions. typedef t(^typeName) (t1,...,tn);

Variable and block declarations. t (^variableName) (t1,...,tn) = ^(t1 variableName1,..., tn variableNamen) { .... }
